Projects > Dropdown

A simple pure CSS drop down menu. I've implemented it within a nav, since I'd imagine it would mostly be used for navigation, but you can easily change it to suit your needs. This example uses CSS3 aimations, and borders. The HTML can get a bit messy, but it's easy once you understand it.

I have also made a vertical menu by changing a few properties, if you want to use that.

I am using an animation, rather than a transition because there is a bug in Chrome that makes text jutter slightly when transitioning opacity.
